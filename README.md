## 프로젝트 설명

- 환율계산 웹 기능 제작

## 프로젝트 실행 방법

#### 1. git clone

```
git clone
```

#### 2. 패키지 설치

```
npm install
```

#### 3. local 프로젝트 실행

```
npm start
```
